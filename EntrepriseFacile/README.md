# Entreprise Facile

Une application mobile React Native pour créer facilement des plans d'affaires professionnels.

## 🚀 Fonctionnalités

### Version Gratuite
- ✅ Création d'un plan d'affaires
- ✅ Modèles de base
- ✅ Support par email
- ✅ Export PDF basique

### Version Premium
- ✅ Plans d'affaires illimités
- ✅ Tous les modèles spécialisés
- ✅ Analyse IA avancée
- ✅ Support prioritaire
- ✅ Export PDF professionnel
- ✅ Intégrations comptables

## 📱 Captures d'écran

L'application propose une interface intuitive avec :
- **Accueil** : Vue d'ensemble et actions rapides
- **Documents** : Gestion de vos plans d'affaires
- **Support** : FAQ et contact client
- **Profil** : Gestion du compte et abonnement

## 🛠️ Installation

### Prérequis
- Node.js (version 20 ou supérieure)
- React Native CLI
- Android Studio (pour Android)
- Xcode (pour iOS)

### Installation des dépendances
```bash
npm install
```

### Lancement sur Android
```bash
npm run android
```

### Lancement sur iOS
```bash
npm run ios
```

## 🏗️ Architecture

```
src/
├── components/          # Composants réutilisables
│   ├── common/         # Composants communs
│   ├── forms/          # Composants de formulaires
│   └── templates/       # Composants de modèles
├── constants/          # Constantes et configuration
├── navigation/         # Configuration de navigation
├── screens/           # Écrans de l'application
│   ├── Auth/          # Authentification
│   ├── Home/          # Accueil
│   ├── Documents/     # Gestion des documents
│   ├── Profile/       # Profil utilisateur
│   └── Support/       # Support client
├── services/          # Services et API
├── types/             # Types TypeScript
└── utils/             # Utilitaires
```
